# Open Weight Tracker

Tracker of open weight models and their performance on European languages and use cases.

Before Europe can depend on open models, we need honest public tracking: what exists, what license it carries, what languages it supports, and how it performs on European tasks.

## What we will track

- Open weight model releases relevant to Europe
- License and deployment constraints
- European language coverage
- Evaluation results from [eurobench](https://github.com/Limes-Labs/eurobench)
- Deployment notes and known limitations

## Status

Early. Structure and first entries are being prepared.

## How to contribute

- Suggest a model to track
- Submit evaluation notes with reproducible methodology
- Help maintain structured data formats
